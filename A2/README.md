# Prerequisite
- Go inside `A2` folder
- First run the jupyter notebook to get the models
- Use those model from `app.py` to use in the UI

# To run the app
- Go inside `A2/code` folder
- Run `python app.py`
- Go to `http://127.0.0.1:8050/` on the browser of your choice
